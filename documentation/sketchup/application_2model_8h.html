)

Creates an overlay and adds it to the model.

Since
SketchUp 2024, API 12.0
Parameters
[in]modelThe model object
[in]infoThe information needed to create the overlay object
[out]overlayThe created overlay object
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if model is an invalid object
  • SU_ERROR_NULL_POINTER_INPUT if info is NULL
  • SU_ERROR_NULL_POINTER_OUTPUT if overlay is NULL
  • SU_ERROR_OVERWRITE_VALID if overlay is already a valid object
  • SU_ERROR_DUPLICATE if the given SUOverlayCreateInfo::id already exists in the model

Releases the overlay after removing it from the model.

Parameters
[in]modelThe model object
[in,out]overlayThe overlay object to be released, will be set to NULL on return.
Since
SketchUp 2024, API 12.0
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if model or overlay is an invalid object
  • SU_ERROR_NULL_POINTER_INPUT if overlay is NULL
  • SU_ERROR_NO_DATA if overlay could be found in the model
